## Sleigh4Rust

A series of common interfaces and functions used by the sleigh-rs projects:
* [sleigh-rs](https://github.com/rbran/sleigh-rs)
* [sleigh2rust](https://github.com/rbran/sleigh2rust)
* [sleigh2macro](https://github.com/rbran/sleigh3macro).
* [sleigh3rust](https://github.com/rbran/sleigh3rust).
* [sleigh3test](https://github.com/rbran/sleigh3test).
